```python
"""InfluxDB v2 writer for solarman_logger.

Converts parsed device readings into InfluxDB Points and writes them.
All numeric values are cast to float to prevent InfluxDB type conflicts (per D-08).
"""
from __future__ import annotations

import logging
from typing import Any

from influxdb_client import InfluxDBClient, WriteOptions
from influxdb_client.client.write_api import SYNCHRONOUS
from influxdb_client.client.exceptions import InfluxDBError

from .config import Config, InfluxConfig

_LOGGER = logging.getLogger(__name__)
```

**Class structure:**

`__init__(self, config: InfluxConfig)`:
- Store `self._config = config`
- Create `self._client = InfluxDBClient(url=config.url, token=config.token, org=config.org)`
- Create `self._write_api = self._client.write_api(write_options=SYNCHRONOUS)` (synchronous write — simple, no batching per D-10 "data dropped, not buffered")
- Store `self._bucket = config.bucket`
- Store `self._org = config.org`

`check_health(self) -> None` (per D-04, D-05):
- Call `health = self._client.ping()`
- If ping returns False or raises, raise `RuntimeError(f"InfluxDB health check failed: cannot reach {self._config.url}")`
- Log info: `f"InfluxDB health check passed: {self._config.url}"`

`write_callback(self, device_name: str, device_type: str, parsed: dict[str, tuple]) -> None` (per D-01, D-02, D-03, D-06, D-07, D-08, D-10):
- Build a dict of fields by iterating `parsed.items()`:
  - For each `(key, (state, value))`: if `isinstance(state, (int, float))`, add `key: float(state)`. If `isinstance(state, str)`, add `key: state`. Skip None values.
- If no fields after filtering, return early (nothing to write).
- Construct Point dict: `{"measurement": device_name, "tags": {"device_name": device_name, "device_type": device_type}, "fields": fields}`
- Wrap write in try/except:
  ```python
  try:
      self._write_api.write(bucket=self._bucket, org=self._org, record=record)
  except Exception as e:
      _LOGGER.warning(f"InfluxDB write failed for {device_name}: {e}")
  ```
  Per D-10: log warning and continue, never re-raise.

`make_data_callback(self, device_configs: dict[str, str]) -> DataCallback`:
- Returns an async function matching `DataCallback = Callable[[str, dict[str, tuple]], Awaitable[None]]`
- The async function looks up `device_type = device_configs[device_name]` and calls `self.write_callback(device_name, device_type, parsed)`
- `device_configs` is a `dict[str, str]` mapping device name to device type, built from `Config.devices`

`close(self) -> None` (per D-11):
- Call `self._write_api.close()` then `self._client.close()`
- Log debug: `"InfluxDB writer closed"`
