    """

Expected YAML structure (per D-03, D-04, D-05):
```yaml
influxdb:
  url: "http://localhost:8086"
  org: "myorg"
  bucket: "solar"
  token: "my-token"

defaults:
  poll_interval: 60  # seconds

devices:
  - name: "Deye Micro"
    host: "192.168.1.100"
    port: 8899           # optional, default 8899
    serial: 1234567890
    profile: "deye_micro.yaml"
    poll_interval: 30    # optional per-device override
    slave: 1             # optional, default 1

  - name: "DDZY Meter"
    host: "192.168.1.101"
    serial: 9876543210
    profile: "ddzy422-d2.yaml"
    # poll_interval not set — uses defaults.poll_interval
    # slave not set — defaults to 1
```

Profile path resolution (per D-05):
- `profile: "deye_micro.yaml"` in config at `/config/config.yaml`
- Resolves to: `profile_dir = "/config/"`, `profile_filename = "deye_micro.yaml"`
